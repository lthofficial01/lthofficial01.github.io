Selected Projects

Seoul Medical Vulnerability Analysis
Compared medical vulnerability across 25 Seoul districts and analyzed demand-adjusted supply gaps for five clinical specialties across 44 administrative neighborhoods in the three highest-vulnerability districts.
Tech Stack: Python, Pandas, scikit-learn, GeoJSON, Folium, Matplotlib
My Role: data collection and preprocessing, geospatial mapping, vulnerability and shortage score design, K-means analysis, and visualization.

Olink Proteomics Analysis Tool
Client-side Olink NPX analysis tool for biomedical data exploration. The project focuses on local browser-based analysis for sensitive data, from preview and pivot to PCA, differential expression, missingness QC, baseline ML, and PPI network analysis.
Tech Stack: React, TypeScript, Pyodide, Web Worker, Python, Plotly
My Role: implemented the client-side Python analysis workflow, Olink format handling, pivot workflow, statistical modules, and interactive visualizations.

SAC on Screen Big Data Pipeline
HDFS, Spark, and Hive pipeline that combines regional age structure and performance reservation patterns to calculate regional genre-fit indicators and Top 3 recommendations.
Tech Stack: HDFS, PySpark, Hive, HiveQL, Bash, Python, Matplotlib
My Role: designed the regional demand index, implemented the PySpark preprocessing pipeline, created Hive analysis queries, and automated result export.
